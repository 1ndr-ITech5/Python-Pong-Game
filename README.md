# Python Pong Game

A classic, retro-style arcade Pong game built in **Python** using the `turtle` graphics library.

## 🎮 Elements & Functionality

*   **Player vs AI:** Play against an autonomous AI opponent.
*   **Controls:** Use `W` (Up) and `S` (Down) to control the left paddle.
*   **Features:** Real-time score tracking, a countdown timer, and automated game-over evaluation (Win/Loss/Draw).

## 🛠️ Components

*   `main.py`: Core game loop and logic coordination.
*   `padels.py`: User paddle controls and AI patrol movement.
*   `ball.py`: Ball physics and bouncing mechanics.
*   `scoreboard.py`: Live score and countdown timer tracking.
*   `field.py` / `functions.py`: Static graphics and game-over rendering.

## ⚙️ How to Run

1. Ensure **Python 3** is installed.
2. Run the game from your terminal:
   ```bash
   python main.py
   ```
